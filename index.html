Theme Liquidator helps operators review deletable themes, shortlist them in the terminal, and confirm every destructive action before anything is removed. Use the steps from the README to install and build the CLI from this project. 
							npm install
							npm run build
							npm install -g
After the global install, run theme-liquidate from any terminal. Start with a dry run so you can review the shortlist before any real deletion is attempted. 
theme-liquidate --shop your-store --dry
 When the dry run looks right, rerun without --dry. 
theme-liquidate --shop your-store
 The CLI opens your hosted Shopify install flow in the browser. The merchant authorises the app. The CLI stores a broker session token locally. 
theme-liquidate auth login --shop your-store
 
theme-liquidate auth list
 
theme-liquidate auth logout
